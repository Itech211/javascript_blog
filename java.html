javascript output

let us dive into javascript outputs

first we can show results in javascript using

    window.alert or alert

    with the alert function the webpage shows the result of the alert as a prompt on the webpage

    for instance we can perform arithmetic functions on two numbers by using alert(num1 + num2) and the page runs the operation

  • sum

    • lets run the sum of 20 and 30 by using the function alert(20 + 30) and see if we will get 30 on the screen prompt



  • division

    • lets run a simple division of 20 by 2 the website prompts 10 on the screen



  • show text

    • we can also change show a text on the screen using alert("text") in javascript

    in summary, the alert function in javascript allows the developer show messages as a prompt on screen
